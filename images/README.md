# Product Images

Place your product images in this folder.

## Recommended Image Names

- `white-eggs.jpg` - White Eggs
- `country-eggs.jpg` - Country Eggs  
- `quail-eggs.jpg` - Quail Eggs
- `milk.jpg` - Milk
- `oil.jpg` - Refined Oil
- `bread.jpg` - Bread
- `biscuit.jpg` - Biscuit

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 400x400 pixels or larger
- Aspect ratio: Square (1:1) works best

## Alternative: Using Image URLs

You can also use image URLs when adding items through the Admin panel. Simply paste the image URL in the "Image URL" field.

## Placeholder Images

If an image is missing, the application will automatically show a placeholder. However, adding actual product images will improve the user experience.
